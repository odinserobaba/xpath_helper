# Часть 2. Запись первого сценария в расширении

Цель: записать простой сценарий "открыть страницу -> нажать кнопку -> проверить текст".

## Шаг 1. Включите инспектор

- На нужной странице нажмите `Alt+X`, либо удерживайте `Ctrl` при наведении
- Выберите элемент
- Нажмите **➕ В список**

## Шаг 2. Добавьте базовые шаги

Рекомендуемый минимальный сценарий:

1. `navigate` — перейти на страницу
2. `click` — нажать кнопку
3. `assert_text` — проверить текст на странице

## Шаг 3. Сделайте шаги понятными

Для каждого шага заполните:

- `title` (кратко: что делает шаг)
- `tags` (например: `smoke`, `auth`, `checkout`)

Это сильно упрощает чтение и поддержку сценариев.

## Шаг 4. Сохраните сценарий

Нажмите **💾 В раннер**.  
Файл будет сохранен в `tests/scenarios/`.

## Пример названий шагов

- `Открыть страницу входа`
- `Нажать кнопку Войти`
- `Проверить заголовок Личный кабинет`
