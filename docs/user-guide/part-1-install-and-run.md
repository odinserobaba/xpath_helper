# Часть 1. Установка и запуск

Цель: поднять локальный раннер и подключить расширение в Chrome.

## Шаг 1. Запустите Web Runner

```bash
cd web-runner
./install.sh
./run.sh
```

Откройте в браузере: `http://127.0.0.1:8000`

## Шаг 2. Установите расширение

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное**
4. Выберите папку `xpath-helper-extension/`

## Мини-проверка

- Открывается страница раннера по адресу `127.0.0.1:8000`
- Расширение видно в списке Chrome Extensions
- Боковая панель расширения открывается по клику

## Если не работает

- Порт занят: остановите другой процесс на `8000` и запустите снова
- Не ставится расширение: проверьте, что выбрана папка `xpath-helper-extension`, а не весь репозиторий
