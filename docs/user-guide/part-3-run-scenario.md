# Часть 3. Запуск сценария в Web Runner

Цель: запустить сценарий из интерфейса и увидеть результат.

## Шаг 1. Выберите сценарий

1. Откройте `http://127.0.0.1:8000`
2. В поле **Сценарий** выберите сохраненный JSON
3. Укажите **Start URL**

## Шаг 2. Запустите

Нажмите **Run**.  
После старта откроется страница вида `/runs/<runId>` с live-статусом.

## Шаг 3. Проверьте результат

- `PASS` — все шаги успешно выполнены
- `FAIL` — на одном из шагов ошибка (смотрите лог и отчет)

## Где смотреть артефакты

- Логи: `tests/logs/`
- Выходные данные раннера: `web-runner/outputs/`

## Типичный поток работы

`записал -> сохранил -> запустил -> посмотрел отчет -> поправил сценарий`
