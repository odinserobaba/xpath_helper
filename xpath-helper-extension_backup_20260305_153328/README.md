# XPath Helper Pro 🔍

Chrome расширение для генерации XPath (Ctrl+Hover режим) - показывает ВСЕ варианты.

## ✨ Возможности

- 🎯 **Ctrl+Hover режим** - активация только при зажатом Ctrl
- 🔍 **ВСЕ уникальные варианты** - без искусственных ограничений
- 👁 **Подсветка элементов** - визуальный outline
- 📋 **Копирование в 1 клик**
- 🔄 **Запасные варианты** - уникальные и неуникальные XPath
- ⚡ **Angular Material** - авто-детекция mat-* компонентов
- 🔥 **Partial Matching** - устойчивость к динамическим классам

## 🚀 Установка

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное**
4. Выберите папку `xpath-helper-extension/`

## 📖 Использование

1. Нажмите на иконку расширения → откроется боковая панель
2. **Зажмите Ctrl** и наведите курсор на элемент
3. Все XPath варианты появятся в панели
4. Нажмите 📋 для копирования или 👁 для подсветки
