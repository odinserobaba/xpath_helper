# Участие в проекте

Спасибо за интерес к **XPath Helper Pro**. Ниже — минимальные правила, чтобы изменения было проще принимать и сопровождать.

## Как предложить изменение

1. Опишите задачу или баг (ожидаемое / фактическое поведение, шаги воспроизведения).
2. Для кода: небольшие PR лучше больших; один PR — одна тема.
3. Сообщения коммитов — **по-английски, в повелительном наклонении**, например: `Add fallback wait for flaky nav`, `Fix set_date on Angular forms`.

## Расширение (`xpath-helper-extension/`)

- После правок перезагрузите расширение на `chrome://extensions/`.
- Убедитесь, что сценарии по-прежнему сохраняются в раннер и формат JSON совместим с `web-runner`.

## Web Runner (`web-runner/`)

- Python: следуйте существующему стилю в `app.py` и смежных модулях.
- После правок UI Flow Editor: `cd web-runner/flow-editor-src && npm ci && npm run build`.
- Локальная проверка: `./run.sh`, затем сценарии и прогон с `tests/scenarios/`.
