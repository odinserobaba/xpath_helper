✦ XPath Helper — Web Runner

Сценарии сохраняются из расширения в tests/scenarios/. Здесь ты видишь список, выбираешь нужный и запускаешь через Python Playwright.

Сценарии

Сохрани сценарий из расширения кнопкой “Сохранить в раннер”, затем нажми ↻.

Запуск

Подстановка переменных: {{baseUrl}}, {{login}} и т.п.
Если в сценарии нет шага navigate, раннеру нужно знать стартовую страницу (в расширении она уже открыта).
Будет доступен как {{baseUrl}}.
Используется для click/input/wait_for_element если у шага нет params.timeoutMs.

Примечание: шаг user_action в веб‑раннере не может ждать пользователя и будет пропускаться с записью в лог.
