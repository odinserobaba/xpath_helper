Сценарии

Сохрани сценарий из расширения кнопкой “Сохранить в раннер”, затем нажми ↻.

Запуск

Подстановка переменных: {{baseUrl}}, {{login}} и т.п.
Если в сценарии нет шага navigate, раннеру нужно знать стартовую страницу (в расширении она уже открыта).
Будет доступен как {{baseUrl}}.
Нужно запустить Chromium-Gost с --remote-debugging-port=9222. Тогда раннер подключится к тому же профилю/окну.
Если включён Attach — headless/slowMo могут не применяться.
В CDP режиме поможет “увидеть” что раннер делает (переключит активную вкладку).
Рисует outline на элементе перед click/input/set_date.
Используется для click/input/wait_for_element если у шага нет params.timeoutMs.
Нужен корректный startUrl — страница как перед этим шагом.

Сравнить два прогона · метрики по сценарию ниже при выборе слева

Если заполнено — сценарий будет выполнен для каждой строки данных (переменные строки перекрывают Variables JSON).
0 = все строки.

Примечание: шаг user_action в веб‑раннере не может ждать пользователя и будет пропускаться с записью в лог.
